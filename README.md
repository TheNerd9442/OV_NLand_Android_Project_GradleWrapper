OV N-Land Network Android Project with Gradle Wrapper

Enthält:
- Minimal Android Studio Projekt
- Gradle Wrapper integriert (gradlew, gradlew.bat, gradle/wrapper)
- Kann direkt auf GitHub hochgeladen werden
- GitHub Actions Workflow kann APK bauen ohne lokale Gradle Installation

Schritte:
1. Zip hochladen zu GitHub
2. Repository öffnen
3. GitHub Actions Workflow starten
4. APK wird automatisch gebaut
